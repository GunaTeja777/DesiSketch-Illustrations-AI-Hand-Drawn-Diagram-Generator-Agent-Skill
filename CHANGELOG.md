# Changelog

All notable changes to the DesiSketch Illustrations skill will be documented in this file.

## [1.1.0] - 2026-07-12
### Added
- Multi-agent compatibility instructions for **Claude Code (CLI)** and **Claude.ai Web UI Projects**.
- Programmatic lint validation test harness at `tests/validate_skill.py`.
- Changelog documentation.

### Changed
- Replaced the original Chinese example illustration assets with English/Hinglish/transliterated Indian visual assets.
- Optimized the YAML frontmatter description in `SKILL.md` to support semantic keyword triggers in Claude Code.
- Cleaned up documentation, links, and filenames across the workspace to make the repository fully organic and agent-agnostic.

## [1.0.0] - 2026-07-11
### Added
- Initial release of the DesiSketch Illustrations skill bundle for Codex/OpenAI agent frameworks.
- Setup parameters (`desisketch-illustrations/SKILL.md`), reference manuals (`references/`), and style-calibration examples.
