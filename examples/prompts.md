# Prompt Examples

Below are prompt examples that can be copied directly and used with the agent.

## 1. Illustration Planning (Shot List Strategy)

```text
Use $desisketch-illustrations Do not generate images yet.
Analyze the article below, identify where visual explanations are needed, and output a shot list of around 5 images.
For each image, specify:
- Placement paragraph
- Theme
- Core metaphor
- Structure type
- Desi's action
- Suggested elements
- Recommended transliterated annotation labels (in Hinglish / Tenglish / Kanglish / English)

<Paste article text here>
```

## 2. Full Image Generation

```text
Use $desisketch-illustrations Generate 4 illustrations for this article.
Requirements: 16:9 horizontal, pure white background, black hand-drawn outline, sparse red/orange/blue Latin-script annotations (e.g., Hinglish or Tenglish transliterated labels).
Each image must explain only one core concept. Do not create complex slides or cute mascot posters.

<Paste article text here>
```

## 3. Long Article Strategy

```text
Use $desisketch-illustrations Plan the illustration strategy for this long article.
Do not distribute illustrations evenly. Only target key cognitive anchors: decision bifurcations, loop closures, before-after state shifts, major pitfalls, or handoff paths.
Target around 6-8 images. Output a shot list first; do not generate any images yet.

<Paste article text here>
```

## 4. Visualizing a Single Statement/Concept

```text
Use $desisketch-illustrations Generate a 16:9 illustration for this concept:

"Trust isn't built by shouting; it is laid down tile by tile with concrete proof."

Make it bizarre but clean. Desi must carry out the physical action. Use 3-5 short Hinglish transliterated labels (e.g., Hinglish: "Saboot", "Bharosa", "Dheere Dheere").
```

## 5. Workflow/Pipeline Theme

```text
Use $desisketch-illustrations Generate a 16:9 illustration for "processing raw input information into three formats: short social posts, newsletters, and video scripts".
Do not draw a formal flowchart or replicate the "one fish, many uses" composition.
Invent a new low-tech physical metaphor where Desi is the central operator driving the process. Use short transliterated labels in Kanglish (e.g., "Kelsa Shuru", "Social Post", "Video Script", "Phala").
```

## 6. Editing: Removing Titles

```text
Use $desisketch-illustrations Edit this image.
Remove the title "Workflow / Process Flowchart" and its underline from the top-left corner, leaving a clean white background.
Do not add or change any other text or objects.
```

## 7. Editing: Increasing Desi's Action Participation

```text
Use $desisketch-illustrations This illustration is on the right track, but Desi feels like a decorative element standing on the side.
Keep the core concept and simple layout, but regenerate a version where Desi is actively driving the mechanics of the metaphor.
Make it more absurd but keep the background pure white, minimalist, and sparse.
```

## 8. Style Calibration Sheets

```text
Use $desisketch-illustrations Generate 5 distinct style calibration illustrations.
Themes: Information Overload, Rapid Prototyping, Compounding Content, One-Person Company, Building Trust.
Each should be generated individually. Do not combine them.
```
